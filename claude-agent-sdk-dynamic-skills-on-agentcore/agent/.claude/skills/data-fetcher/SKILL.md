# Data Fetcher Skill

## Description
Fetches data from various sources for analysis.

## Parameters
- `source`: Data source (api, database, file)
- `query`: Query parameters

## Dependencies
None

## Usage
```python
result = fetch_data(source="api", query="sales_data")
```

## Output
Returns structured data ready for processing.
